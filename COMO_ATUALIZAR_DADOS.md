# Como atualizar os dados

## Ensalamento
```bash
npm run atualizar:ensalamento -- caminho/do/arquivo.json
```

## Eventos
```bash
npm run atualizar:eventos -- caminho/do/arquivo.json
```

## FAQ
```bash
npm run atualizar:faq -- caminho/do/arquivo.json
```

O script atualiza somente a seção correta dentro de `src/data/dados.json`.
No FAQ, a categoria é preenchida automaticamente como `Geral`.
